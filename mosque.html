Shah Jalal Mosque Website


OVERVIEW:

This website was built for a local mosque to help with the prayers time table and other activities the mosque offers.

DESIGN:

We chose an Islamic mosaic in the header of the page since most of the mosques are decorated with this geometrical patterns. We choose bright colours that combine nicely with the mosaic.



FUNCTIONALITY:

For the prayers time table we created a database with the prayer times in .pdf then we convert them in .json and write a javascript that will show the right time in the website.


The code for this website is open source! You can download it from my GitHub repository: https://github.com/aframelia/euston-mosque
